#  Elite Driving School Management System

A desktop-based driving school management application built using **JavaFX, Hibernate, and MySQL**, designed to streamline lesson bookings, user management, and payment tracking for a driving school.

---

##  Project Overview

The **Elite Driving School System** allows students to register, book driving lessons, and track their booking and payment status. Managers can efficiently manage users, bookings, and payments through a centralized system.

---

##  Features

###  User (Student)
- Register and login securely
- Book driving lessons
- View booking details and schedule
- Check payment status
- Track lesson history

---

###  Manager (Admin)
- Secure admin login
- Manage users (Create, Update, Delete)
- Manage driving lesson bookings
- Update booking status (Pending / Approved / Completed)
- Manage payment records and statuses
- Full control over system data

---

## Tech Stack

-  Java
-  JavaFX (UI)
-  Hibernate (ORM)
-  MySQL (Database)
-  JDBC / Hibernate Config
-  Layerd Architecture

---

##  Database Modules

- Users Table
- Bookings Table
- Payments Table
- Lessons Table

---

##  System Workflow

1. User registers and logs in
2. User books a driving lesson
3. Booking is sent to manager dashboard
4. Manager approves/rejects booking
5. Payment status is updated
6. User can track all updates in real-time
